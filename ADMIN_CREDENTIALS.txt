========================================
  API ENDPOINTS
========================================

Base URL: http://localhost:5000

Admin Authentication:
  POST /api/admin/login
  GET  /api/admin/profile (requires auth)

Analytics (all require admin token):
  GET /api/analytics/dashboard
  GET /api/analytics/users/stats
  GET /api/analytics/users/activity
  GET /api/analytics/downloads/stats
  GET /api/analytics/downloads/history

Download Tracking (requires user token):
  POST /api/analytics/track-download
